generateQRcode.c:
To generate the URI and barcode, type $ ./generateQRcode issure account_name secret(Hex)
First, we read the secret key value from the input secret_hex and store this value byte by byte in myByteArray, every 2 characters of the secret_hex consisting of an element of myByteArray. Then, we use myByteArray and base32_encode() to generate the secret. Finally, we format the information and print it out.

validateQRcode.c:
To validate the HTOP and TOTP generated by the first part, type $ ./validateQRcode secret(Hex) HOTP TOTP
This validator computes both the HOTP and TOTP from the same algorithm: HMAC = H[(K ¨’ opad) || H((K ¨’ ipad) || M)], and the difference is that the HOTP used counter = 1 and TOTP used counter = the current UNIX time.
